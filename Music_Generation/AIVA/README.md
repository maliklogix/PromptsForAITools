# AIVA — Prompt Guide

## Overview
AIVA (Artificial Intelligence Virtual Artist) is an AI music composition tool specialized in creating emotional soundtracks, cinematic scores, classical compositions, and game music. Recognized as a virtual composer by SACEM, AIVA uses deep learning trained on thousands of classical compositions to generate original orchestral and multi-genre music.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` to configure AI composition workflows or scoring tools.
2. Adapt it for any AI-powered soundtrack or orchestral music generation pipeline.

### User Prompts
- Use AIVA's interface at **aiva.ai** to generate compositions from genre/style presets.
- Upload reference tracks or MIDI files for style guidance.
- Specify instrumentation, duration, tempo, and emotional intent.

## Tips for Best Results
- **Specify the use case** — film, game, commercial, or concert performance all require different approaches.
- **Describe the emotional arc** — "starts tense, builds to triumphant" is more useful than just "epic."
- **Mention orchestration** — tell AIVA which instruments to feature and which to exclude.
- **Set precise duration** — especially important for media scoring (15s, 30s, 60s, etc.).
- **Reference classical styles** — "in the style of Debussy" or "Romantic-era piano" helps dial in the aesthetic.
- **Request stems** — ask for separated instrument tracks for mixing flexibility.
