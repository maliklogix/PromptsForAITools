# Mubert — Prompt Guide

## Overview
Mubert is a real-time AI generative music platform that creates endless, unique, royalty-free soundtracks. Unlike traditional AI music generators that produce individual tracks, Mubert generates continuous, non-repeating musical streams that adapt in real-time to context, mood, and user input. It's built for integration into apps, games, streams, and products via API.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` to configure generative music workflows or build adaptive music products.
2. Adapt it for designing real-time music generation systems for apps and platforms.

### User Prompts
- Use Mubert at **mubert.com** to generate tracks by selecting mood, genre, and duration.
- Integrate via the Mubert API for real-time generative music in your product.
- Use Mubert Studio to create custom generative streams with fine-tuned parameters.

## Tips for Best Results
- **Define the context** — workout, meditation, gaming, and streaming all need fundamentally different music.
- **Specify the energy map** — describe how energy should change over time for dynamic content.
- **Use mood tags** — combine multiple tags like "calm + dreamy + nocturnal" for precise results.
- **Set tempo ranges** — a BPM range gives the generator room to create natural variation.
- **Think in streams, not tracks** — Mubert excels at continuous music, leverage that.
- **Plan for transitions** — if your app has states, define how music should transition between them.
