## 🌟 What Is This?

**PromptsForAITools** is the ultimate open-source library of production-ready **system prompts** and **user prompts** for the most popular AI tools on the market. Whether you're a developer, marketer, designer, data analyst, or AI enthusiast — this repo gives you battle-tested prompts to unlock the full potential of every AI tool you use.

> 💡 **Stop writing mediocre prompts.** Use expert-crafted, structured prompts that get you 10x better results from any AI tool.

| 🧠 **Expert-Level Quality** | System prompts are 500+ words with clear roles, rules, and reasoning strategies |

Every tool folder contains **exactly 3 files**, following a consistent, easy-to-use format:

## 🚀 How to Use

### Quick Start

1. **Navigate** → Go to the category and tool folder you need
2. **Copy** → Paste the `System_Prompt.txt` content into your tool's system/custom instructions
3. **Prompt** → Use `User_Prompts.txt` as templates and adapt them to your use case
4. **Learn** → Read the tool-specific `README.md` for tips on getting the best results

### For Developers (API Usage)

```python
# Example: Using a system prompt with OpenAI API
import openai

# Load the system prompt
with open("Coding_Assistants/ChatGPT/System_Prompt.txt", "r") as f:
    system_prompt = f.read()

response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": "Your task here..."}
    ]
)
```

### For Non-Developers

| Step | Where | How |
|---|---|---|
| 1️⃣ Open `System_Prompt.txt` | Your AI tool's settings/custom instructions | Copy-paste the entire content |
| 2️⃣ Open `User_Prompts.txt` | The chat/prompt input field | Copy a prompt, customize the details |
| 3️⃣ Iterate | Same chat session | Refine the output by giving specific feedback |
